🖥️ View (Vista)

📌 Descripción

La vista es la encargada de la interacción con el usuario mediante consola. Muestra información y recibe entradas.



🧠 Responsabilidades

* Mostrar menús
* Mostrar datos (videojuegos, usuarios, etc.)
* Solicitar información al usuario
* Mostrar mensajes (errores, confirmaciones)



🧩 Componentes

* `VistaMenu` → Menú principal
* `VistaUsuario` → Interacción con usuarios
* `VistaVideojuego` → Mostrar catálogo



🔄 Ejemplo de funciones

* `mostrarMenu()`
* `mostrarCatalogo()`
* `pedirDatosUsuario()`



📌 Notas

* No contiene lógica de negocio
* No accede directamente a la base de datos
