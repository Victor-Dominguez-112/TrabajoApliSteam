🎮 Tienda de Videojuegos MVC (C++)

📌 Descripción

Este proyecto consiste en el desarrollo de una tienda de videojuegos tipo Steam utilizando el lenguaje C++ y la arquitectura Modelo-Vista-Controlador (MVC). El sistema permite gestionar usuarios, visualizar videojuegos y realizar compras, utilizando una base de datos MySQL mediante XAMPP.



🧱 Arquitectura del sistema

El proyecto sigue el patrón MVC:

* **Model:** Manejo de datos y conexión a la base de datos
* **View:** Interfaz de usuario (consola)
* **Controller:** Lógica del sistema



📂 Estructura del proyecto

```
/TiendaVideojuegosMVC
│
├── model/
├── view/
├── controller/
└── main.cpp
```



⚙️ Tecnologías utilizadas

* C++
* MySQL (XAMPP)
* MySQL Connector/C++ (opcional)



🚀 Funcionalidades

* Registro de usuarios
* Visualización de catálogo
* Compra de videojuegos
* Consulta de biblioteca



▶️ Ejecución

1. Iniciar XAMPP (Apache y MySQL)
2. Configurar la base de datos
3. Compilar el proyecto en C++
4. Ejecutar `main.cpp`
